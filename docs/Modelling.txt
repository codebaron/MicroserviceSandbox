
I. RentalInventory Domain

		2. Rent a rental property

		4. Invoice a rental property
		
		5. Add notes to a rental property
		
		6. Add/update/revoke security access code
		
		7. Add/remove delinquent status
		
		8. Add/remove property from temporary hold
		
		9. Link/unlink Customer Loyalty account
		
		10. Add/remove promotion to rental property
		
		11. Apply payment
	
	B. Value objects
	
		1. Invoice & invoice line items (may originate in billing domain)
		
		2. Promotion (originate in promotion domain)
		
		3. Customer loyalty account (originate in customer loyalty domain)
